About

The person
behind the screens.

I'm Kashish Gupta — a Game UI/UX Designer who's spent the last year designing interfaces for games that players actually ship, download, and play. I work at Apex Dream Studio remotely, handling everything from concept to final delivery.

I started as a Unity developer intern — writing game logic, setting up UI systems — before moving fully into design. That background means I understand what I'm handing off. I know how a button state works in code & what breaks in Unity.

I've designed for mobile, AR, and TV — card games with premium gold UI, AR fashion experiences, spooky casual games, and interactive mall installations. I also sketch, build low-poly worlds in Blender, and occasionally code small games from scratch.

Currently pursuing my BCA from Lovely Professional University alongside full-time work. Open to Game UI/UX Designer roles, onsite or remote.
